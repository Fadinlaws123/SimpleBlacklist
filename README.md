## 📖 About

**SimpleBlacklist** is the original SimpleDevelopments multi-blacklist resource for restricting vehicles, weapons, and player ped models on FiveM servers.

Each blacklist system can be enabled or disabled independently and the restricted models are managed through the resource configuration.

This repository is the original version of the project. A newer **SimpleBlacklistV2** repository is also available with expanded configuration and bypass controls.

---

## ✨ Features

- Vehicle blacklist
- Weapon blacklist
- Ped model blacklist
- Individually toggleable blacklist systems
- Simple spawn-name based configuration
- Standalone with no framework dependency

---

## ⚙️ Configuration

Edit `Config.lua` to enable or disable each blacklist and add the appropriate GTA V spawn names for the vehicles, weapons, or ped models you want to restrict.

When adding multiple entries to a Lua table, remember to separate entries with commas.

---

## 📥 Installation

1. Place `SimpleBlacklist` in your server's resources directory.
2. Configure the blacklist entries in `Config.lua`.
3. Add the resource to your `server.cfg`.
4. Restart the resource or server.

---

## 🌐 SimpleDevelopments

SimpleBlacklist is maintained as a legacy **SimpleDevelopments** resource for servers that still use the original system.
